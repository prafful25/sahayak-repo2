PURPOSE OF THIS REPOSITORY
------------------------------------------------------------
This repository contains all API specs that require a NEW domain,
a NEW Docker container, and a NEW SSL certificate on the VPS.
These are net-new product verticals not covered by the existing 12 APIs.
Each new domain = a new product line with its own identity and ICP.
Build order: provision domain → deploy FastAPI container → configure Nginx
             → Let's Encrypt SSL → add OpenCLAW scraper → billing.

BUILD STRATEGY
============================================================================
Each domain in this repo requires the following one-time setup:
  1. Register subdomain on Hostinger DNS panel (5 min per domain)
  2. Deploy new FastAPI container via Docker Compose (30 min per domain)
  3. Configure Nginx reverse proxy entry + Let's Encrypt SSL (15 min)
  4. Set up OpenCLAW scraper mission for the data source
  5. Connect billing via existing Razorpay integration
